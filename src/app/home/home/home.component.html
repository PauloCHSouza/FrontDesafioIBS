Esta é a página inicial do meu desafio.

Este projeto consiste em uma aplicação que integra o front-end e o back-end, sendo o back-end compartilhado via GitHub com o nome "DesafioIBS".

Sobre o Back-end:

  • NestJS: Um framework Node.js para construção de aplicações server-side eficientes e escaláveis.
  • Prisma: Uma ferramenta ORM (Object-Relational Mapping) moderna para Node.js e TypeScript.
  • class-transformer: Uma biblioteca para transformar objetos, garantindo a integridade dos dados.
  • class-validator: Uma biblioteca para validação de dados baseada em decorators.
  • RxJS: Uma biblioteca para programação reativa em JavaScript.

Este back-end foi estruturado com o NestJS, seguindo boas práticas de desenvolvimento e utilizando ferramentas como o Prisma para interação com o banco de dados. Os dados são validados utilizando as bibliotecas class-transformer e class-validator, garantindo integridade e consistência dos dados.

Sobre o Front-end:

  • Angular: Um framework JavaScript/TypeScript mantido pelo Google para construção de aplicações web modernas e responsivas.
  • Bootstrap: Um framework front-end para design responsivo e componentes web.
  • Font Awesome: Uma biblioteca de ícones para utilização em projetos web.
  • ngx-mask: Uma biblioteca Angular para aplicação de máscaras em campos de formulário.
  • ng-bootstrap: Uma biblioteca Angular para utilização dos componentes do Bootstrap de forma nativa no Angular.
  • rxjs: Uma biblioteca para programação reativa em JavaScript.
  • Express: Um framework web para Node.js que fornece um conjunto robusto de recursos para aplicativos web e móveis.
